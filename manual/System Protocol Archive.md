# 📘 Lab Operational Codex: System Protocol Archive
## Workspace Purge & Clarity Protocol (v1.1.0)

研究作業中にウィンドウが氾濫し、思考のノイズが発生した際の「緊急浄化プロトコル」を以下に定義する。

1. **Operation: Instant Clarity (画面の即時整理)**
   - **The Soloist Sweep**: `Win` + `Home`
     - 効果: アクティブな画面以外、背後のウィンドウをすべて最小化。
   - **Total Cloak**: `Win` + `D`
     - 効果: すべてを隠し、デスクトップを表示する（再度押すと復元）。
   - **Focus Lock**: `Alt` + `Tab`
     - 効果: 氾濫したウィンドウの中から必要な一つを瞬時に選び出す。

2. **Operation: Hard Reset (ウィンドウの強制終了)**
   - **Process Execution**: `Alt` + `F4`
     - 効果: 現在開いているウィンドウを即座にクローズする。
   - **Tab Obliterator**: `Ctrl` + `Shift` + `W`
     - 効果: ブラウザやエディタ内のすべてのタブを一括で閉じる。
   - **Group Termination**: **タスクバー右クリック**
     - 効果: 特定アプリ（フォト等）の全ウィンドウを一括終了。

3. **Operation: Environment Recovery (環境の復元)**
   - **Ghost Recall**: `Ctrl` + `Shift` + `T`
     - 効果: 誤って閉じてしまったブラウザのタブを、直前のものから順に蘇生する。
