# ラボ・クイックリファレンス (v1.0.0)

## 1. メンテナンス用語（AIとの対話用）
博士が「全部を書き潰す」不安なく、安全にマニュアルを育てるための知的な合成術。

* **パッチ（Patch）を当てる**
    * ファイルの「古くなった一部」だけを外科的に書き換えること。全体を壊さず、改善点だけを反映させる技。
* **マージ（Merge）する**
    * 二つの異なる指示や知見（例：午前と午後の決定事項）を、大事なところを残して一つに統合すること。

## 3. 運用フロー（相対実行版）
1. 画像を `input/` へ配置。
2. ルートの `up.bat` を実行（Python同期 → JS統合エンジン起動）。
3. `output/` または `data/` に生成された結果を Gemini へ報告。
※ 個別の画像診察が必要な場合は `node vision_core/ファイル名.js` を実行。

## 4. ドキュメントの役割分担（鉄則）
* **operation_manual.md**: AI（ジェミニ）が正確に動くための「詳細設計図」。
* **lab_quick_reference.md**: 博士が瞬時に確認するための「虎の巻」。
機能,ショートカットキー,備考
行の削除,Ctrl + Shift + K,1行丸ごと消去します。
行のコピー（複製）,Shift + Alt + ↓ (または ↑),現在の行をすぐ下（または上）にコピーします。
行の切り取り,Ctrl + X,選択範囲がなくても、カーソルがある行を丸ごと切り取ります。
行の貼り付け,Ctrl + V,切り取り・コピーした行を現在のカーソル位置に挿入します。
