# Fallout 76 自動解析ラボ 運用指令書 v1.8.6

## 1. 目的
本ドキュメントは、Fallout 76 のデータ抽出・原子分解プロトコルを定義し、プロジェクトのポータビリティ（相対パス管理）と安全性を確保するためのものである。

## 2. ディレクトリ構造（相対パスの原則）
全てのスクリプトは、プロジェクトルートを起点とした相対パスで動作する。
- `./` : プロジェクトルート（up.bat, xpr.bat）
- `./script/` : Python/PowerShell等の基幹スクリプト
- `./script/bin/` : Node.js実行エンジン群
- `./vision_core/` : 【新設】精密診断・OCR検証用JS群
- `./data/` : 解析用JSONデータ
- `./input/` : 解析対象のスクリーンショット
- `./output/` : 処理結果・切り抜き画像

## 3. 実行承認プロトコル
- **二段階承認**: `y/n` による博士の最終確認を必須とする。
- **相対実行**: バッチファイルは必ずダブルクリック、またはルートディレクトリから呼び出すこと。

## 4. 博士流エイリアス（原子分解版）
抽出データは `integration_engine.js` および `subtitle_generator.py` を通じ、以下の変換を自動適用する。
- 鋭い視線 → **Per増加**
- サヴェージ・ストライク → **アーマー貫通**
- アンステイブル → **素早く・爆発**
- ウエストテック研究センター → **ウエ研**
- スーパーミュータント → **スパミュ**
