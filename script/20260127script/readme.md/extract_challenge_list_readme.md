Extract Challenge List Script
このスクリプトは、Fallout 76 のスクリーンショット（WQHD/2560x1440）から、デイリーオプスやチャレンジリスト部分を正確に切り出し、別フォルダへ出力するためのプログラムです。

⚙️ 環境要件
Node.js: インストール済みであること

sharp: 画像処理ライブラリ

未導入の場合はターミナルで以下を実行してください：

Bash
npm install sharp
📂 ディレクトリ構成
スクリプト内で以下の絶対パスが指定されています。

Input: D:\nvidia_captures\afterburner_png_stills

Output: D:\nvidia_captures\output

Note: 実行後も入力元のオリジナルファイルは削除されず保持されます。

📍 抽出設定 (WQHD用)
アップロードされた最新の座標セットに基づいています。

left: 530

top: 55

width: 690

height: 1150

🚀 実行方法
VS Codeのターミナル、またはコマンドプロンプトを開きます。

以下のコマンドを実行してください：

Bash
node extract_challenge_list.js
⚠️ 注意事項
入力ファイル名（inputFileName 変数）は、処理したいファイルに合わせて手動で書き換えてください。

Cursor のAI機能を使用せず、トラブルシューティングや設計変更が必要な場合は、すべてGeminiとの対話を通じて解決してください。

🌟 これからできること（拡張案）
今後のワークフロー保全と知識補填のための、機能拡張のアイデアです。博士の指示に基づき、必要に応じて実装を検討します。

出力フォルダの自動生成: 保存先フォルダがない場合にエラーを出さず、自動で作成する機能。

タイムスタンプ付与: 出力名に時刻（例：_102530）を付加し、過去のデータを上書きせず保存する機能。

解像度の自動チェック: 画像がWQHD（2560x1440）か確認し、誤った範囲の切り出しを防ぐ安全機能。

処理完了の通知: 成功時にWindows通知を表示したり、自動で出力フォルダを開く利便性向上。

複数ファイルの一括処理: ファイル名をリスト化し、一回の実行で複数枚を連続して抜き出す機能。
