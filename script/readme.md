# 博士の Fallout 76 ラボ運用マニュアル v1.6.2

## 1. 概要
Gemini から供給される統合 JSON データを、Python 経由で各 JS システムへ原子分解して供給する。

## 2. キーワード変換リスト (v1.6.2)
デイリーオプスの説明文から以下のキーワードを検出し、博士流エイリアスに変換する。

| 説明文のキーワード | 博士流エイリアス | 備考 |
| :--- | :--- | :--- |
| 素早く / 移動速度 | **素早く** | 「高速」から変更。 |
| 死亡時に爆発 | **爆発** | 不安定（アンステイブル）要素。 |
| Perception / 大幅に増加 | **Per増加** | 鋭い視線の効果。 |
| 防御力を無視 | **アーマー貫通** | サヴェージ・ストライクの効果。 |
| 近接攻撃でしか | **弾力性** | 博士指定の正式名称。 |
| 体力を回復 | **回復** | グループ再生の効果。 |
| 透明 / 見えなくなる | **透明** | ステルス系の効果。 |

## 3. 特殊ルール
- **ダブル変異**: 説明文から複数のキーワードを抽出し「・」で繋ぐ（例：素早く・爆発）。
- **アトミックショップ**: 1st 限定品には必ず `status: "Fallout 1st"` を付与する。

## 4. 処理フロー
1. Gemini で画像解析 -> `Fallout76_data.json` を生成。
2. `data/` フォルダへ配置。
3. `python script/clip_magic.py` を実行して原子分解。
