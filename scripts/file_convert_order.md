Fallout76-Lab 運用指令書 (v1.9.4 - layout-optimized)

1. 生成・出力形式
以下の3つの形式を順番に出力せよ。

① マークダウン・サマリー: 
   - 第1見出しを「【本日の天気とイベント】」とし、ミネルヴァ情報をそのセクションの末尾に統合せよ。
   - その直後の1行目に「皆様おはようございます YYYY年M月D日 曜日 [天候のみ]」と記載せよ（天候は福岡のデータを使用）。
   - 第2見出しを「【本日のオプス】」とせよ。
   - 第3見出しを「【本日の無料アイテム】」とし、[一般無料] または [1st専用無料] のラベルを各情報の頭に付与せよ。
   - 第4見出しを「【本日のお買い得】」とし、[お買い得] ラベルを付与せよ。
   - 第5見出しを「【Fallout 1st 専用】」とし、[1st専用お買い得] ラベルを付与せよ。
   - 変異名の後のカッコ書き、ショップの終了期限、引用タグは一切出力しないこと。
   - 末尾に「【本日のデイリーチャレンジ】」の全リストを含めること。

② JSON オブジェクト: 
   - free_items (label含む), deals (label含む) を分けた構造で出力せよ。

③ ダウンロードファイル: 
   - 命名規則（Fallout76_YYYY_MM_DD_XX.json）を厳守せよ。

2. 情報収集・抽出ルール（博士流エイリアス）
- ロケーション: ウエストテック研究センター → ウエ研
- 敵派閥: スーパーミュータント → スパミュ
- 変異: 鋭い視線 → Per増加、サヴェージ・ストライク → アーマー貫通、不安定 → 爆発、氷の手 → 氷結、スティングフロスト → 氷結・毒
- [cite_start]情報の引き算: カッコ書きや余計な注釈は徹底排除せよ。
